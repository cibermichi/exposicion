Introducción

Esta página está creada únicamente con HTML, sin aplicar hojas de estilo. Más adelante se añadirá CSS para mejorar su presentación y mostrar la diferencia entre el contenido estructurado y el contenido estilizado.

Preguntas a Resolver

1. ¿A qué se denomina Multimedia, hipermedia e hipertexto?

Multimedia combina texto, imagen, audio y video. Hipermedia integra varios medios conectados por enlaces interactivos. Hipertexto es un texto con enlaces que permiten navegar entre diferentes secciones o documentos.

2. ¿Qué es una página web y qué editores se usan para crearla?

Una página web es un documento digital que se visualiza mediante un navegador. Se puede crear con HTML y CSS, usando editores como Visual Studio Code, Sublime Text o Notepad++.

3. ¿Qué significa HTML y para qué se utiliza?

HTML (HyperText Markup Language) es el lenguaje que estructura el contenido de una página web.

4. ¿Qué etiquetas marcan el inicio y final de un documento HTML?

El documento inicia con <html> y termina con </html>.

5. ¿Qué indican las etiquetas <body> y </body>?

Encierran todo el contenido visible de la página, como texto, imágenes y tablas.

6. ¿Cuáles etiquetas se declaran dentro de <head>?

Se incluyen <title>, <meta>, <link> y <style>.

7. ¿Cuáles son las etiquetas de los atributos de texto?

<b> (negrita), <i> (cursiva), <u> (subrayado), <mark> (resaltado).

8. ¿Cuál es la etiqueta que participa como parámetro en otras etiquetas?

<img> o <a> utilizan atributos como src y href para enlazar recursos.

9. ¿Cuáles son las formas de utilizar las etiquetas de listas?

  • <ul>: lista no ordenada
  • <ol>: lista ordenada
  • <li>: elemento de la lista

10. ¿Cuál sería la forma de colocar un color de fondo?

Con CSS: body { background-color: lightgreen; }

11. ¿Para qué se utiliza la etiqueta <font>?

Se usaba para cambiar color y tamaño de texto, pero hoy se reemplaza con propiedades CSS como color y font-family.

12. ¿Qué diferencia hay entre <p> y <div>?

<p> define un párrafo, mientras que <div> agrupa elementos para organizar el diseño.

13. ¿Cuál es la estructura básica de un código HTML?

<!DOCTYPE html>
<html>
  <head>
    <title>Mi página</title>
  </head>
  <body>
    Contenido visible
  </body>
</html>

14. ¿Cuáles son las etiquetas de encabezado?

Van de <h1> a <h6>, de mayor a menor jerarquía.
