Introducción

Esta página demuestra la diferencia entre HTML puro y HTML con CSS. Aquí se aplican estilos visuales para mejorar la presentación, la organización y la experiencia del usuario.

Preguntas a Resolver

¿A qué se denomina Multimedia, hipermedia e hipertexto?

Multimedia combina texto, imagen, audio y video. Hipermedia conecta varios medios mediante enlaces interactivos. Hipertexto es texto con enlaces que permiten saltar entre páginas o secciones.

¿Qué es una página web y qué editores se usan para crearla?

Una página web es un documento digital que se visualiza en un navegador. Se crea con HTML y CSS, usando editores como Visual Studio Code, Sublime Text o Notepad++.

¿Qué significa HTML y para qué se utiliza?

HTML (HyperText Markup Language) estructura el contenido de las páginas web.

¿Qué etiquetas marcan el inicio y final de un documento HTML?

Comienza con <html> y termina con </html>.

¿Qué indican las etiquetas <body> y </body>?

Contienen todo el contenido visible de la página, como texto, imágenes y tablas.

¿Cuáles etiquetas se declaran dentro de <head>?

<title>, <meta>, <link> y <style>.

¿Cuáles son las etiquetas de los atributos de texto?

<b> (negrita), <i> (cursiva), <u> (subrayado), <mark> (resaltado).

¿Cuál es la etiqueta que participa como parámetro en otras etiquetas?

<img> o <a> utilizan atributos como src y href.

¿Cuáles son las formas de utilizar las etiquetas de listas?
  • <ul>: lista no ordenada
  • <ol>: lista ordenada
  • <li>: elemento de la lista
¿Cuál sería la forma de colocar un color de fondo?

Con CSS: body { background-color: lightgreen; }

¿Para qué se utiliza la etiqueta <font>?

Antes se usaba para cambiar color y tamaño del texto, hoy se reemplaza con CSS (color y font-family).

¿Qué diferencia hay entre <p> y <div>?

<p> crea párrafos, <div> agrupa elementos para organizar el diseño.

¿Cuál es la estructura básica de un código HTML? 
<!DOCTYPE html>
<html>
  <head>
    <title>Mi página</title>
  </head>
  <body>
    Contenido visible
  </body>
</html>
¿Cuáles son las etiquetas de encabezado?

<h1> a <h6> son títulos jerárquicos. Ejemplo: <h1>Fabricio Andrés Zeballos Leigue</h1>
